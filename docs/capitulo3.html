Capítulo 3: CSS

O que é CSS?

CSS (Cascading Style Sheets) é uma linguagem de estilo que define a apresentação visual das páginas da web.

Tipos de códigos em CSS

Em HTML temos os seguintes tipos de códigos:

  • Setores de Elementos
  • Seletores de classes
  • seletores de ID
  • Seletores de Pseudo-classes
  • Regras de estilo em bloco
  • Estilo inline
  • Estilo Externo

Setores de Elementos: Você pode aplicar estilos diretamente a elementos HTML usando seletores de elementos.

Seletores de Classes: Os seletores de classes permitem aplicar estilos a elementos com uma classe específica. Você define a classe no HTML usando o atributo class.

Seletores de ID: Os seletores de ID permitem aplicar estilos a elementos com um ID específico. Você define o ID no HTML usando o atributo id.

Seletores de Pseudo-classes: As pseudo-classes permitem estilizar elementos com base em seu estado ou posição.

Regras de estilo em bloco: Você pode agrupar várias regras de estilo em um bloco usando chaves { }. Isso é útil quando você deseja aplicar o mesmo estilo a vários seletores.

Estilo inline: Você também pode aplicar estilos diretamente a um elemento HTML usando o atributo style.

Estilo Externo: É comum definir estilos em um arquivo CSS externo e vinculá-lo à sua página HTML usando a tag link. Isso permite que você mantenha seus estilos separados do conteúdo HTML.
