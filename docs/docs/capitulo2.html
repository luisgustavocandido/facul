Capítulo 2: Tags HTML

O que é uma TAG HTML?

Uma "tag" HTML (ou elemento HTML) é um componente fundamental da linguagem de marcação HTML (Hypertext Markup Language). As tags HTML são usadas para estruturar e formatar o conteúdo de uma página da web. Elas atuam como marcadores que informam ao navegador como exibir e interpretar diferentes partes do conteúdo.

Quais são os tipos de TAG que existem?

Tags de Estruturação e Conteúdo:

  • html
  • head
  • title
  • body
  • h1 até h6
  • etc...

Tags de Formulário:

  • form
  • button
  • select
  • etc...

Tags de Midia:

  • audio
  • video
  • iframe
  • canvas

Tags Semânticas

  • section
  • article
  • header
  • nav
